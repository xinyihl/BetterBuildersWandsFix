## BetterBuildersWandsFix
Go to [Releases](https://github.com/xinyihl/BetterBuildersWandsFix/releases/latest) to download the release version. Go to [Actions](https://github.com/xinyihl/BetterBuildersWandsFix/actions) to download the development version.

### Features:
*   Fixed the issue of losing NBT data when placing & withdrawing blocks.
*   Fixed the issue of missing tags when checking if an item exists in the inventory.
*   Fixed resolved compatibility issues with the Custom Selection Box
*   Added the ability to place off-hand items when an item is present in the off-hand.
*   Added Ctrl+Z to undo the last placement.
*   Added break mode functionality and related crafting recipe
*   Allow multiple undo block placements
*   Bypass light update when placing nbt-free blocks
